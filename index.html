Red Pitaya - RadioBox application

This repository enhances the RedPitaya hardware and the ecosystem by add-ons for Transmitting and Receiving radio signals.

It is intented to:

  • expand the FPGA features as well as
  • creating an own radio_box application

to the RedPitaya ecosystem.
